# Introducción

Este repositorio contiene una colección de proyectos desarrollados con **JavaScript moderno (ES6+)**, diseñados para fortalecer habilidades prácticas en programación web. Los proyectos incluyen ejemplos de manipulación del DOM, consumo de APIs, asincronía, almacenamiento local y lógica de programación.  

Cada proyecto busca ofrecer un reto real para aplicar conceptos clave de JavaScript, fomentando la comprensión de buenas prácticas y el desarrollo de soluciones interactivas y dinámicas.
